HTML 문법

(1) 요소 :  내용을 포함한 태그 전체를 요소라고 한다.
ex) <h1>제목</h1> 에서 제목과 태그 모두를 합친 것이 요소이다.

(2) 속성
- 태그에 추가로 정보를 제공하거나 태그의 동작이나 표현을 제어할 수 있는 설정값을 의미
- 이름과 값으로 이루어져 있고, 이름과 값 사이의 공백을 허용하지 않는다.
ex) <h1 id="title">제목</h1>

(3) 태그 중첩
- 태그 안에 또 다른 태그를 사용하는 것
- 태그는 중첩을 통해 부모 태그와 자식 태그의 관계가 만들어진다.

(4) 빈 태그
- 시작 태그만 존재하고 종료 태그가 존재하지 않는 것
ex) <br>

(5) 공백 = 두 칸 이상의 공백과 개행을 모두 무시한다.
ex) 아래는 모두 같은 결과가 나온다.
<h1>Hello, HTML</h1>
<h1>Hello,     HTML</h1>
<h1>
    Hello,
    HTML
</h1>
=> Hello, HTML

(6) 문서의 기본 구조
<!DOCTYPE html>
<html lang="ko">
    <head>
        <meta charset="UTF-8">
        <title>HTML</title>
    </head>
    <body>
        <h1>Hello, HTML</h1>
    </body>
</html>

* <!DOCTYPE html>
- 태그는 아니고 이 문서가 어떤 버전으로 작성되었는지 브라우저에게 알려주는 선언문
-  예시 코드의 <!DOCTYPE html>은 이 문서가 html5로 작성되었음을 알려준다.
- 최상단에 위치해야 하고 어떤 것도 <!DOCTYPE html>보다 먼저 와서는 안된다.

* <html lang="ko"> 
- 해당 문서가 어느 언어로 작성되었는지를 알려준다.

* <head></head> (헤드 태그)
- 브라우저 화면에 표시되지는 않고 문서의 기본 정보, 설정이나 외부 stylesheet나 js를 연결하는 역할

* <meta charset="UTF-8">
- 문자의 인코딩 방식을 지정

* <body></body>
- 실제 브라우저 화면에 나타나는 내용
